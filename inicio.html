¿Qué es la accesibilidad digital?

Las personas con discapacidad visual pueden usar tecnologías como lectores de pantalla y teclados especiales para navegar por internet de forma autónoma.

Lectores de Pantalla

Software que transforma texto en voz o braille para facilitar la navegación.

Teclados Adaptados

Con teclas grandes y contraste para facilitar la escritura.

Audio Guías

Retroalimentación sonora y descripciones para entender contenido visual.
